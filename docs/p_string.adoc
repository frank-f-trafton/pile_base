== API


=== pString.ptn_dollar

A pattern for https://www.lua.org/manual/5.1/manual.html#pdf-string.gsub[**string.gsub**], matching a dollar sign followed by any combination of ASCII letters, numbers and underscores.

* `$FOO_bar123`

'''


=== pString.ptn_percent

A pattern for https://www.lua.org/manual/5.1/manual.html#pdf-string.gsub[**string.gsub**], matching a two percent signs surrounding any combination of ASCII letters, numbers and underscores.

* `%FOO_bar123%`
