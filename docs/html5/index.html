PILE Base is a utility library for Lua, versions 5.1 - 5.4.

  • Git repository

Contributors

  • Frank F. Trafton

PILE Base includes code from:

  • lume by rxi

  • LUIGI by airstruck

Code from FFT’s previous libraries

  • Utf8Tools was included in 2024 (as the UTF-8 modules).

  • StringWalk (formerly StringReader) was included in 2026.
